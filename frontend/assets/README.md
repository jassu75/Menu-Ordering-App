# Assets

Place your app icon and splash screen here:
- `icon.png` — 1024x1024 app icon
- `splash.png` — 1284x2778 splash screen (or similar)
- `adaptive-icon.png` — 1024x1024 for Android adaptive icon

Expo will use these automatically based on app.json configuration.

For development, Expo provides default assets if these are missing.
