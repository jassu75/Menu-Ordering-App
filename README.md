# 🍽️ Intelligent Bistro

AI-powered food ordering application built with React Native and TypeScript. Users can browse menu items, chat with an AI assistant for recommendations, manage their cart, and complete secure payments using Stripe.

---

## 📱 Overview

Intelligent Bistro combines conversational AI with a modern food ordering experience.

Users can:

- Chat with an AI assistant to discover food items
- Add and manage items in their cart
- Place secure orders using Stripe
- Enjoy a smooth cross-platform mobile experience

---

## ✨ Features

- 🤖 AI-powered food recommendation assistant
- 🛒 Smart cart management
- 💳 Secure Stripe payment integration
- ⚡ Fast React Native mobile interface
- 🧠 Zustand state management
- 🔥 Full TypeScript support

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React Native, TypeScript, Zustand |
| Backend | Node.js, Express.js, TypeScript |
| Payments | Stripe |
| AI | Groq API |

---

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/jassu75/Menu-Ordering-App.git
cd Menu-Ordering-App
```

---

# ⚙️ Environment Variables

Create a `.env` file inside the `backend` directory.

```env
PORT=3001

FRONTEND_URL=your_frontend_url

GROQ_API_KEY=your_groq_api_key

STRIPE_SECRET_KEY=your_stripe_secret_key
```

---

# 🔧 Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# 📱 Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

# 🤖 Integrations

This project uses:

- Groq API for AI-powered food ordering assistance
- Stripe for secure online payments
